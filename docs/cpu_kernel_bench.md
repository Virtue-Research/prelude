# CPU Kernel Benchmark: cpu_ops vs sgl-cpu vs candle vs oneDNN

**CPU**: AMD Ryzen 9 9950X 16-Core (Zen5, AVX-512, no AMX)
**NUMA**: 16 threads pinned to node 0 physical cores
**Date**: 2026-03-11

> Speedup ratios shown as `(Nx)` = slower/faster relative to cpu_ops.
> sgl-cpu extend_attention skipped: SEGFAULT on non-AMX CPUs.
> sgl-cpu GEMM skipped for M ≤ 4: SEGFAULT on non-AMX CPUs.
